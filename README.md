User Greetings Drupal Module
============================

By Micah Courey, 20-Oct-2015

A module for Drupal that displays custom splash screens when a user logs in or out, based on their user role.

Setup
----------
* Clone repository:
```console
$ git clone https://github.com/micahcourey/user_greetings-.git
```
* move the module directory to your Drupal site
```console
$ mv user_greetings /sites/all/modules
```
* Enable the User Greetings module in the Drupal Modules menu.
